## 📝 核心算法说明

### 1. GridCableRouter - A*寻路
```
功能：在网格上寻找单根电缆的最短路径
算法：A* with 8-direction movement
优化：Octile distance 启发式
```

### 2. TopologyExtractor - 拓扑提取
```
功能：从布线路径中提取关键点和骨架
方法：图论 + 关键点合并
优化：基于路径距离的合并策略
```

### 3. MILP Optimizer - 全局优化
```
功能：找到最优树形线束
模型：混合整数线性规划
求解器：SCIP
优化：割平面约束（建议）
```

### 4. SPT Heuristic - 启发式
```
功能：快速近似最优解
算法：最短路径树枚举
优化：Demand-aware 版本
```
