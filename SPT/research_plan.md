## 📚 研究方向

### 方向 1: MILP 求解加速

**当前状态**: v2.0 已实现 Warm Start + 割平面 + 预处理

**深入研究**:
1. **Benders 分解**
   - 主问题：拓扑选择
   - 子问题：路径规划
   - 预期加速：10-20x

2. **Lagrangian 松弛**
   - 松弛复杂约束
   - 次梯度优化
   - 预期加速：5-10x

3. **列生成 (Column Generation)**
   - 动态生成路径变量
   - 适用于大规模问题
   - 预期加速：20-50x

4. **并行求解**
   - 多线程分支定界
   - 分布式计算
   - 预期加速：硬件相关

### 方向 2: 转弯半径约束

**当前状态**: v2.0 已实现显式约束 + 改进 A*

**深入研究**:
1. **Dubins 路径集成**
   - 精确的最短路径计算
   - 考虑最小转弯半径
   - 应用于后处理

2. **Reeds-Shepp 曲线**
   - 支持倒车场景
   - 更灵活的路径规划
   - 适用于复杂环境

3. **微分几何方法**
   - 曲率连续约束
   - 高阶平滑性
   - 适用于高精度场景

### 方向 3: 启发式算法改进

**当前状态**: 已实现 GA/PSO/SA/VNS 等

**深入研究**:
1. **混合元启发式**
   - GA + VNS
   - PSO + SA
   - 发挥各自优势

2. **大规模邻域搜索 (LNS)**
   - 破坏 + 修复策略
   - 适用于超大规模问题

3. **学习增强启发式**
   - 用 ML 预测好的初始解
   - 强化学习指导搜索方向

### 方向 4: 机器学习应用

**当前状态**: 尚未开始

**研究方向**:
1. **图神经网络 (GNN)**
   - 学习电缆布线模式
   - 端到端路径预测
   - 快速生成可行解

2. **强化学习 (RL)**
   - 训练智能体进行路径规划
   - 适应动态环境
   - 持续学习优化

3. **迁移学习**
   - 在小规模问题上训练
   - 迁移到大规模问题
   - 减少计算成本

### 方向 5: 实际应用扩展

**当前状态**: 基于园区网络布局

**扩展方向**:
1. **3D 布线**
   - 考虑高度维度
   - 多层建筑布线
   - 管道/电缆共架

2. **动态障碍物**
   - 实时重规划
   - 移动障碍物避让
   - 时间维度规划

3. **多目标优化**
   - 成本 + 可靠性 + 可维护性
   - Pareto 前沿求解
   - 决策支持

4. **不确定性建模**
   - 需求不确定性
   - 成本波动
   - 鲁棒优化
