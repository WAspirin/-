### DQN/Q-Learning 核心要点 (Day 8 - 2026-03-08):
- **核心公式**: Q(s,a) = Q(s,a) + α * [r + γ*max_a' Q(s',a') - Q(s,a)]
- **状态空间**: 100 (10x10 网格位置)
- **动作空间**: 4 (上/下/左/右)
- **奖励设计**: 每步 -0.1, 到达 +10, 碰撞 -1, 超时 -5
- **探索策略**: ε-greedy, 从 1.0 衰减到 0.01 (系数 0.995)
- **测试结果**: 300 回合训练，最佳奖励 10.00, 测试平均步数 24.4
- **收敛分析**: 前 50 回合探索，50-150 学习策略，150-300 稳定收敛

| 适应性 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ |

### PPO 核心要点 (Day 9 - 2026-03-09):
- **Actor-Critic 架构**: Actor 学习策略 (动作概率)，Critic 学习价值 (状态评估)
- **GAE 优势估计**: δ_t = r_t + γ*V(s_{t+1}) - V(s_t)，λ=0.95 平衡偏差 - 方差
- **策略梯度更新**: 使用优势函数加权梯度，A(s,a) = Q(s,a) - V(s)
- **截断机制**: 限制策略更新幅度 (ε=0.2)，防止训练崩溃
- **测试结果**: 15x15 网格，500 回合，到达率~80-90%，路径长度~25-35 步
- **PPO vs DQN**: PPO 是 Policy-based (直接学策略)，DQN 是 Value-based (学 Q 值)；PPO 更稳定，支持连续动作

**Week 2 进度**: 2/7 完成 (29%) - DQN ✅, PPO ✅

### SW-RDQN 完整复现 (Day 10 进阶 - 2026-03-11):
**论文**: 《基于关键路径点的深度强化学习复合路径规划》(Sun et al. 2026)

**核心创新**:
1. **阶段关键点机制 (Stage-Keys)**: 多阶段子目标分解，将长路径分解为多个关键点序列
2. **轨迹预测模块**: RNN 预测动态障碍物轨迹，提前规避
3. **优先经验回放 (论文式 19)**: 基于奖励+TD 误差的加权采样
   - 优先级公式：P(i) = w_reward * log(reward_i + |min| + 1) + w_td_error * log(|TD_error_i| + 1)
4. **网络结构 (论文图 4)**: CNN + LSTM + Dueling DQN
   - 空间特征：双层 CNN (5x5→32ch, 3x3→64ch 下采样)
   - 时序建模：2 层 LSTM (hidden=256, dropout=0.3)
   - Dueling 分支：Value + Advantage，Q = V + (A - mean(A))

**奖励函数 (论文式 16)**:
- R_total = R_goal + R_align + R_avoid + R_pre + R_step
- R_goal: 到达目标 +100
- R_align: 轨迹对齐奖励 (cos 相似度)
- R_avoid: 避障奖励 (距离分级：-3/-2/-1)
- R_pre: 预测落点靠近惩罚 (-1.25 * distance)
- R_step: 每步惩罚 -0.01

**代码实现**:
- `20_swr_dqn_paper_implementation_fixed.py`: 完整修复版 (~500 行)
- `SW_RDQN_optimized.py`: 优化版本
- 修复问题：OMP 库冲突、3D 输入维度、状态序列管理

**测试结果**: 24x24 网格，4 时间步，6 空间特征，10 回合训练验证

### SW-RDQN 完整复现版 v3.0 (2026-03-10) 🎯

**论文**: Sun et al. (2026). Deep reinforcement learning-based composite path planning with key path points. IEEE Transactions on Robotics.

**核心架构** (三层规划系统):
1. **全局规划层**: Voronoi 图关键点提取 (Algorithm 1)
   - 安全距离剪枝 (d_safe=2.0)
   - 起点/终点投影到骨架
   - 关键点简化 (移除共线点)

2. **局部控制层**: SW-RDQN 网络 (CNN + GRU + Dueling DQN)
   - **输入**: 11x11 观测窗口 × 5 通道 × 4 帧滑动窗口
   - **CNN 编码器**: 3 层卷积 (32→64→128 filters) + BatchNorm
   - **GRU 时序**: 2 层，hidden_size=256，dropout=0.3
   - **Dueling 头部**: 价值流 + 优势流，输出 8 方向动作 Q 值

3. **底层规划器**: A* 精确路径 (关键点间连接)

**加权优先经验回放** (式 17-19):
- 优先级：priority = (|TD_error| + 1e-6)^α, α=0.6
- 采样权重：w = (N * P(i))^(-β) / max(w)，β从 0.4 递增到 1.0
- 重要性采样校正，消除偏差

**奖励函数设计**:
- r_goal = +100 (到达子目标)
- r_obstacle = -50 (碰撞) / -10 (距离<2)
- r_dynamic = -30 (碰撞动态障碍)
- r_smooth = +5 (平滑动作) / -2 (转向)
- r_step = -0.1 (每步惩罚)
