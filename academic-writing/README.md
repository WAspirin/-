## 📖 核心内容

### 论文结构 (IMRaD)
```
Title (标题)
├── Abstract (摘要)
├── Introduction (引言)
├── Methods (方法)
├── Results (结果)
├── Discussion (讨论)
├── Conclusion (结论)
└── References (参考文献)
```

### 写作要点
- **标题**: 简洁、准确、有吸引力
- **摘要**: 结构化，包含背景、方法、结果、结论
- **引言**: 背景→问题→贡献
- **方法**: 可复现、详细、清晰
- **结果**: 图表配合、数据支撑
- **讨论**: 解释结果、对比文献、说明局限
