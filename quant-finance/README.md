## 🚀 快速开始

### 安装依赖

```bash
# 基础库
pip install pandas numpy matplotlib

# 技术分析
pip install TA-Lib

# 数据获取
pip install yfinance akshare

# 回测框架
pip install backtrader

# 机器学习
pip install scikit-learn xgboost
```

### 运行第一个示例

```bash
cd examples
python 01_data_fetch.py
```
