## 📚 技能 6: Best Minds

### 核心概念
**Best Minds** = 最佳思维模型集合，帮助 AI 做出更好的决策。

### 核心内容

#### 1. 决策框架
```
问题定义 → 信息收集 → 方案生成 → 评估选择 → 执行反馈
```

#### 2. 思维模型库
- **第一性原理**: 从基本原理推导
- **逆向思维**: 从结果倒推原因
- **系统思维**: 整体而非局部
- **概率思维**: 考虑不确定性
- **机会成本**: 考虑放弃的价值

#### 3. 问题解决策略
- **分解**: 大问题→小问题
- **类比**: 类似问题类似解
- **抽象**: 提取本质特征
- **迭代**: 逐步改进

### 应用场景

#### 1. 算法选择
```
问题特征 → 匹配思维模型 → 选择算法
例：
- 最短路径 → 第一性原理 → Dijkstra
- 大规模优化 → 分解思维 → 层次化 DRL
- 动态环境 → 概率思维 → RL
```

#### 2. 研究决策
```
研究方向 → 评估标准 → 优先级排序
标准：
- 创新性
- 实用性
- 可行性
- 影响力
```

### 应用到电缆布线

#### 决策点
1. **算法选择**
   - 小规模 → MILP (精确最优)
   - 中规模 → VNS/TS (质量好)
   - 大规模 → DRL (适应性强)

2. **参数调优**
   - 实验设计
   - 响应面分析
   - 贝叶斯优化

3. **问题诊断**
   - 根因分析
   - 5 Whys
   - 鱼骨图

- **标签搜索**: 基于分类

#### 2. 技能评估
- **质量评分**: 代码质量、文档完整度
- **使用统计**: 下载量、活跃度

#### 3. 推荐系统
- **基于内容**: 相似技能推荐
- **基于协作**: 用户行为推荐

#### 1. 技能索引
```python
class SkillIndex:
    def __init__(self):
        self.skills = []
        self.embeddings = []
    
    def add_skill(self, skill):
        # 提取特征
        features = extract_features(skill)
        # 向量化
        embedding = embed(features)
        # 存储
        self.skills.append(skill)
        self.embeddings.append(embedding)
    
    def search(self, query, top_k=10):
        # 向量搜索
        similar = search_similar(self.embeddings, query, top_k)
        return [self.skills[i] for i in similar]
```
