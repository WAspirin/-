## 🔍 学习方法

### 步骤 1: GitHub 搜索
```bash
# 搜索模板
https://github.com/search?q={skill-name}+agent+skills&type=repositories
```

### 步骤 2: 仓库分析
对于每个找到的仓库:
1. 阅读 README.md
2. 查看代码结构
3. 分析核心实现
4. 记录关键设计

### 步骤 3: 学习笔记
为每个技能创建学习笔记:
- 功能描述
- 架构设计
- 核心代码
- 可以借鉴的点

### 步骤 4: 实践应用
- 哪些可以用到电缆布线优化项目
- 哪些可以改进现有技能
- 哪些可以创建新技能
