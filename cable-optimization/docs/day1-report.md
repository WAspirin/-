### 1. MILP 基础示例 (`examples/01_milp_basic.py`)

**问题**：简单网络流优化

**模型**：
- 决策变量：流量 x_e (连续) + 边选择 y_e (0-1)
- 目标：最小化布线成本
- 约束：流量守恒 + 容量限制

**技术栈**：
- PuLP (建模)
- CBC (求解器)
- NetworkX (图论)
- Matplotlib (可视化)

**运行结果**：
```
✓ 创建示例网络：6 个节点，9 条边
✓ 布线任务：从 s 到 t, 流量需求 = 50
✓ 找到最优解！
  最优总成本：$35.00
  使用的边数：3
```

**关键收获**：
- 掌握了 MILP 建模基本流程
- 理解了流量守恒约束的写法
- 学会了使用 PuLP 调用 CBC 求解器

### 2. Dijkstra 算法 (`examples/02_dijkstra.py`)

**问题**：单源最短路径

**实现**：
- 优先队列优化版本
- O((V+E)logV) 时间复杂度
- 路径重构功能

**运行结果**：
```
✓ 创建电缆网络：8 个节点，13 条边
✓ 源点：J0

从 J0 到各节点的最短距离:
  J1: 距离 = 3.0, 路径：J0 → J2 → J1
  J2: 距离 = 2.0, 路径：J0 → J2
  J3: 距离 = 4.0, 路径：J0 → J2 → J1 → J3
  ...
```

**关键收获**：
- 深入理解了贪心策略
- 掌握了优先队列的使用
- 实现了最短路径树可视化

## 📞 给 WonderXi 的建议

基于今日学习，对您的研究方向有以下建议：

### MILP 方法
1. **求解器选择**：建议使用 Gurobi（学术许可免费），比 CBC 快 10-100 倍
2. **建模技巧**：
   - 使用紧的上界约束
   - 添加对称性破缺约束
   - 考虑 Lazy Constraint 处理复杂约束

### 最短路径树
1. **扩展方向**：
   - 考虑多目标（成本 + 可靠性）
   - 加入容量约束（带资源约束的最短路径）
   - 动态环境（实时重规划）

### 启发式算法
1. **混合策略**：GA + 局部搜索效果通常优于纯 GA
2. **参数调优**：建议使用实验设计 (DOE) 系统调参
3. **对比基准**：建立标准测试集，公平对比算法
