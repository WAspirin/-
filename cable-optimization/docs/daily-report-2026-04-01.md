## 📚 学习内容

### 1. 综合布线系统标准

**参考标准**:
- GB 50311-2016《综合布线系统工程设计规范》
- TIA/EIA-568《商业建筑电信布线标准》

**6 大子系统**:
1. 进线间子系统 (External Connection)
2. 设备间子系统 (Equipment Room)
3. 垂直干线子系统 (Backbone Cabling)
4. 水平布线子系统 (Horizontal Cabling)
5. 工作区子系统 (Work Area)
6. 管理子系统 (Administration)

### 2. 电缆选型规则

| 带宽需求 | 距离 ≤55m | 距离 ≤100m | 距离 >100m |
|----------|-----------|------------|------------|
| ≤1Gbps | Cat5e | Cat5e/Cat6 | 光纤 |
| ≤10Gbps | Cat6 | Cat6A/Cat7 | 光纤 |
| >10Gbps | - | - | 光纤 OM3/OS2 |

**关键约束**:
- 水平布线最大长度：90m (GB 50311)
- 工作区跳线：≤5m
- 设备跳线：≤5m
- 总通道长度：≤100m

### 3. 电磁干扰 (EMI) 防护

**典型干扰源安全距离**:
| 干扰源 | 功率 | 最小平行走线距离 |
|--------|------|-----------------|
| 低压电源线 (<380V) | - | 130mm |
| 电梯电机 | 高 | 2000mm |
| 空调变频器 | 中 | 500mm |
| 微波炉 | 高 | 1000mm |
| 无线电设备 | 高 | 300mm |

**防护措施**:
- 金属线槽屏蔽
- 使用 FTP/STP 电缆
- 保持安全距离
- 交叉时垂直穿越

### 4. 3D 空间建模

建筑布线是真正的 3D 问题：
- X/Y: 平面坐标
- Z: 楼层 (层高通常 3.5m)
- 垂直布线通过专用线井 (Riser)

**距离计算**:
```python
def distance_to(self, other):
    dx = self.x - other.x
    dy = self.y - other.y
    dz = (self.z - other.z) * 3.5  # 层高
    return np.sqrt(dx**2 + dy**2 + dz**2)
```

## 🎯 关键技术洞察

### 1. 分层架构的优势

智能建筑布线采用分层设计：
```
设备间 (ER) → 电信间 (TR) → 工作区 (WA)
    ↓            ↓           ↓
  主配线      分配线      信息插座
```

**优势**:
- 模块化：每层独立，便于维护
- 可扩展：新增设备只需连接到最近 TR
- 故障隔离：单层故障不影响其他层
- 管理清晰：层次化的标识系统

### 2. 干扰建模与规避

将干扰源建模为圆形区域：
```python
@dataclass
class InterferenceZone:
    center: Position3D
    radius: float      # 影响半径
    severity: float    # 严重程度 (0-1)
    
    def affects_position(self, pos) -> bool:
        return self.center.horizontal_distance(pos) <= self.radius
```

**干扰评分**:
```python
score = severity × (1 - normalized_distance)
```

路径规划时选择干扰评分最低的路由。

### 3. 电缆成本优化

**成本模型**:
```python
cable_costs = {
    CableCategory.CAT5E: 3.0,      # 元/米
    CableCategory.CAT6: 5.0,
    CableCategory.CAT6A: 8.0,
    CableCategory.CAT7: 12.0,
    CableCategory.FIBER_OM3: 25.0,
    CableCategory.FIBER_OS2: 40.0,
}
```

**优化策略**:
- 在满足性能前提下选择最经济的电缆
- 水平布线尽量短 (距离影响成本和性能)
- 垂直干线使用光纤 (高带宽，长距离)

### 4. PoE 供电集成

现代智能建筑中，PoE (Power over Ethernet) 广泛应用：
- WiFi AP: 802.3af/at (15.4W/30W)
- 摄像头：802.3at (30W)
- IP 电话：802.3af (15.4W)

**优势**:
- 单电缆同时传输数据和电力
- 简化布线，降低成本
- 集中供电管理
- UPS 保护更简单


### 问题 2: 3D 路径可视化

**挑战**: 如何在 2D 图上表示 3D 布线路径

**解决**: 
- 平面图：只显示同层路径，垂直路径用特殊标记
- 3D 视图：使用 mpl_toolkits.mplot3d 绘制真实 3D 路径
- 系统分析：多维度图表展示统计信息

### 问题 3: 干扰区域与路径冲突

**问题**: 某些设备位置靠近干扰源

**解决**:
- 在电缆选型时考虑干扰 (使用屏蔽电缆)
- 路径规划时增加干扰惩罚项
- 工程上建议调整设备位置
