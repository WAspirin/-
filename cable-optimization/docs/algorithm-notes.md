## 📐 一、数学规划法

### 1.1 MILP (混合整数线性规划)

**核心思想**：将问题建模为线性目标函数 + 线性约束，部分变量为整数

**标准形式**：
```
min c^T x
s.t.  Ax ≤ b
      x ≥ 0
      x_i ∈ Z (部分变量)
```

**布线问题建模**：
- **决策变量**：
  - `x_e`: 边 e 上的流量（连续）
  - `y_e`: 是否使用边 e（0-1 整数）
  
- **目标函数**：最小化总成本 `min Σ c_e * y_e`

- **约束**：
  1. 流量守恒：流入 = 流出（除源汇点）
  2. 容量限制：`x_e ≤ cap_e * y_e`
  3. 整数性：`y_e ∈ {0,1}`

**求解器**：
- Gurobi (商业，最快)
- CPLEX (商业)
- CBC (开源，PuLP 内置)
- SCIP (开源)

**优缺点**：
- ✅ 保证全局最优
- ✅ 理论成熟
- ❌ 计算复杂度高（NP-hard）
- ❌ 大规模问题求解难

### 1.2 最短路径树方法

#### Dijkstra 算法

**核心思想**：贪心策略，每次选择距离最小的未访问节点

**伪代码**：
```
dist[source] = 0, 其他为∞
PQ = 优先队列 [(0, source)]

while PQ 非空:
    (d, u) = PQ.pop()
    if u 已访问：continue
    
    for v in neighbors(u):
        new_dist = dist[u] + weight(u,v)
        if new_dist < dist[v]:
            dist[v] = new_dist
            prev[v] = u
            PQ.push((new_dist, v))
```

**时间复杂度**：O((V+E)logV)

**适用场景**：
- 单源最短路径
- 边权非负
- 需要精确解

#### A* 算法

**改进**：加入启发式函数 h(n)

**评估函数**：`f(n) = g(n) + h(n)`
- `g(n)`: 从起点到 n 的实际代价
- `h(n)`: 从 n 到终点的估计代价（如欧氏距离）

**关键**：启发式函数必须是可采纳的（不高估实际代价）

## 🔥 二、启发式算法

### 2.1 遗传算法 (GA)

**灵感**：生物进化论

**核心操作**：
1. **编码**：路径 → 染色体（排列）
2. **选择**：锦标赛/轮盘赌
3. **交叉**：OX/PMX
4. **变异**：交换/逆转

**参数调优**：
- 种群大小：50-200
- 交叉率：0.7-0.9
- 变异率：0.01-0.2
- 精英保留：5-10%

**优缺点**：
- ✅ 全局搜索能力强
- ✅ 并行性好
- ❌ 收敛速度慢
- ❌ 参数敏感

### 2.2 粒子群优化 (PSO)

**灵感**：鸟群觅食

**速度更新**：
```
v = w*v + c1*r1*(pbest-x) + c2*r2*(gbest-x)
x = x + v
```

- `w`: 惯性权重
- `c1, c2`: 学习因子
- `pbest`: 个体最优
- `gbest`: 全局最优

### 2.3 模拟退火 (SA)

**灵感**：金属退火

**Metropolis 准则**：
```
如果 ΔE < 0: 接受新解
否则：以概率 exp(-ΔE/T) 接受
```

**退火计划**：
- 初始温度：高
- 降温系数：0.9-0.99
- 终止温度：接近 0

### 2.4 变邻域搜索 (VNS)

**核心思想**：系统性地改变邻域结构

**流程**：
```
for k = 1 to k_max:
    扰动：在当前解的第 k 个邻域随机扰动
    局部搜索：在新邻域内搜索
    如果改进：接受并重置 k=1
    否则：k = k+1
```

**Q-learning 更新**：
```
Q(s,a) = Q(s,a) + α * [r + γ*max_a' Q(s',a') - Q(s,a)]
```

**DQN 改进**：
- 经验回放
- 目标网络
- Dueling 架构

**布线问题建模**：
- **状态**：当前图状态 + 已选路径
- **动作**：选择下一条边
- **奖励**：-成本（每步），+ 奖励（完成）

### 3.3 图神经网络 (GNN)

**优势**：直接处理图结构

**应用**：

## 💡 五、实战技巧

### MILP 建模技巧

1. **紧化约束**：尽可能使用紧的上界
2. **对称性破缺**：添加约束减少对称解
3. **Lazy Constraint**：动态添加约束（如子环消除）
4. **Warm Start**：用启发式解初始化

### 启发式调参

1. **多样性 vs 收敛**：变异率平衡
2. **精英保留**：防止退化
3. **自适应参数**：根据搜索状态调整
4. **混合策略**：GA+ 局部搜索

### 强化学习训练

1. **奖励塑形**：稀疏奖励→稠密奖励
2. **课程学习**：从简单到复杂
3. **迁移学习**：预训练 + 微调
4. **多智能体**：分布式探索

## 📅 Day 2 新增内容 (2026-03-03)

### 2.2 粒子群优化 (PSO) - 详细实现

**算法流程**:
```
1. 初始化粒子群（随机位置和速度）
2. 评估每个粒子的适应度
3. 更新个体最优 (pbest) 和全局最优 (gbest)
4. 更新速度：v = w*v + c1*r1*(pbest-x) + c2*r2*(gbest-x)
5. 更新位置：x = x + v
6. 重复 2-5 直到收敛
```

**关键参数**:
- `w` (惯性权重): 0.4-0.9，平衡全局/局部搜索
- `c1` (个体学习因子): 通常 1.5-2.0
- `c2` (群体学习因子): 通常 1.5-2.0
- 种群大小：30-50

**线缆布线应用**:
- 优化路径控制点坐标
- 连续变量优化
- 可加入障碍物惩罚项

**代码示例**: `examples/04_pso.py`

### 2.3 模拟退火 (SA) - 详细实现

**Metropolis 准则**:
```
如果 ΔE < 0 (新解更好): 接受
否则：以概率 exp(-ΔE/T) 接受
```

**退火计划**:
- 初始温度：高（如 1000-10000）
- 降温系数：0.9-0.995
- 终止温度：接近 0（如 1e-6）
- 每温度迭代次数：50-200

**邻域操作**:
- 连续问题：高斯扰动
- 离散问题（TSP）：交换、逆转

**优缺点**:
- ✅ 实现简单，参数少
- ✅ 理论上能以概率 1 找到全局最优
- ✅ 适合组合优化
- ❌ 收敛速度慢
- ❌ 参数敏感

**线缆布线应用**:
- 路径控制点优化
- TSP 类回路优化
- 可处理离散和连续变量

**代码示例**: `examples/05_simulated_annealing.py`

### 1.3 A* 搜索算法 - 详细实现

**评估函数**: `f(n) = g(n) + h(n)`
- `g(n)`: 从起点到 n 的实际代价
- `h(n)`: 从 n 到终点的启发式估计

**常用启发式**:
| 启发式 | 公式 | 适用场景 |
|--------|------|----------|
| 欧氏距离 | √(dx²+dy²) | 任意方向移动 |
| 曼哈顿距离 | |dx|+|dy| | 四方向移动 |
| 切比雪夫距离 | max(|dx|,|dy|) | 八方向移动 |
| Octile 距离 | max+ (√2-1)*min | 八方向精确估计 |

**可采纳性**: 启发式必须不高估实际代价，才能保证找到最优路径

**搜索效率**:
- 时间复杂度：O(b^d)，b 为分支因子，d 为深度
- 空间复杂度：O(b^d)，需要存储 OpenSet

**线缆布线应用**:
- 网格环境路径规划
- 避开障碍物
- 机房/工厂布局布线

**代码示例**: `examples/06_astar.py`
