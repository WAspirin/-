## 🚨 异常处理

### 如果任务未执行
1. 检查 Gateway 状态：`openclaw gateway status`
2. 查看 cron 日志：`openclaw cron runs`
3. 手动触发：`openclaw cron run <task-id>`

### 如果超时
- 默认超时：2 小时
- 自动终止并通知
- 可调整：`openclaw cron edit <task-id> --timeout-seconds <new-value>`

### 如果通知失败
- 检查飞书配置
- 查看 Gateway 日志
- 手动发送总结
