# 线缆优化 - 每日学习任务

## ⏰ 定时任务配置

**时间**: 每天早上 9:00 (Asia/Shanghai)  
**频率**: 每天一次  
**时长**: 预计 1-2 小时

## 📋 学习流程

### 1. 检查进度 (5 分钟)
- 阅读 `docs/learning-progress.md`
- 确认下一个要学习的算法
- 查看待办事项

### 2. 算法学习 (30 分钟)
- 阅读相关论文/文档
- 理解算法原理
- 记录关键公式和步骤

### 3. 代码实现 (45 分钟)
- 创建新的示例文件 `examples/XX_algorithm_name.py`
- 实现完整算法
- 添加详细注释
- 编写可视化代码

### 4. 文档更新 (20 分钟)
- 更新 `docs/algorithm-notes.md`
- 编写 `docs/daily-report.md`
- 更新 `docs/learning-progress.md`

### 5. 提交代码 (10 分钟)
```bash
cd /root/.openclaw/workspace
git add cable-optimization/
git commit -m "feat: 实现 [算法名称] - Day X"
git push origin master
```

## 🎯 每日交付物

每个学习日后，仓库应包含：

2. **学习笔记** (`docs/`)
